## Docker

To start the frontend in a Docker contatiner, run these commands:<br> 
`sudo docker build -t fedboozgrunt .` <br>
`sudo docker run -p 9000:9000 -v <THIS DIRECTORY>:/usr/src/app/client/grunt/ -d fedboozgrunt`<br>
